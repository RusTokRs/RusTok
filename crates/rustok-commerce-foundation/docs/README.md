# Документация `rustok-commerce-foundation`

`rustok-commerce-foundation` — общий support crate для распиленного commerce-контура.

## Что сейчас внутри

- общие DTO;
- общие SeaORM entities;
- единый error surface;
- общие search helpers.

## Кто использует

- `rustok-product`
- `rustok-pricing`
- `rustok-inventory`
- `rustok-commerce`
