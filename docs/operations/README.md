# Operations Documentation

This directory contains Athanor-facing operational documentation pages for runtime configuration keys, environment variables, scripts, deployment resources, and runbook targets discovered from the RusToK repository.

The pages are bounded, source-linked read surfaces for diagnostics and automation checks. They document canonical operational entities with source evidence so agents do not need to inspect generated Athanor JSONL directly.
